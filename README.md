# Mobius SDK

Shared client utilities for the Mobius Organizational Intelligence Operating System.

## Mission

Reduce duplicated integration logic across Mobius services by centralizing common helpers for events, identity context, Pulse instrumentation, Memory clients, Repository DNA, and API access.

## Enterprise Alpha Scope

- Event client helpers
- Pulse metric helpers
- Memory event helpers
- Repository DNA helpers
- Identity context helpers
- API request helpers

## Rule

Common service integration logic belongs in the SDK before it is duplicated across repositories.
